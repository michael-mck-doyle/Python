
myReference for various terms used in Python



Modules - A file containing Python definitions and statements is called a python module.
 So naturally, the file name is the module name which is appended with the suffix .py
 - Python module index - https://docs.python.org/3/py-modindex.html
 - execute command to check available modules: "help('modules')" from the Python shell

Package - Python package is a collection of python modules.  Python module is a single python file whereas python
 package is a directory having multiple python scripts and __init__.py file defining the package details.
 To install a Python package use... e.g. pip install --upgrade certifi
